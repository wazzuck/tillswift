Tillswift A5 flyer — 2-up on A4, cut in half

Two identical A5 flyers print on each A4 sheet. Print double-sided, then cut down the dashed centre line to get two finished flyers.
  1. File → Print (or use the button below)
  2. Paper size: A4 · Orientation: Landscape · Two-sided / Duplex: On (long-edge OR short-edge — either works, both halves carry identical content)
  3. Margins: None · Background graphics: On · Scaling: Default / 100%
  4. Print, let the duplex finish, then cut along the green dashed line down the middle
